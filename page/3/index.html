Single Number 给定一个非空整数数组，除了某个元素只出现一次以外，其余每个元素均出现两次。找出那个只出现了一次的元素。 说明 你的算法应该具有线性时间复杂度。 你可以不使用额外空间来实现吗？ 示例 1: 12输入: [2,2,1]输出: 1 示例 2: 12输入: [4,1,2,1,2]输出: 4 解题 由于每个相同的元素都会出现两次，又因为两个相同的元素的异或为0。 a^a = 0 并且异或存在交换律，因此 aba = b 因此本题中直接将所有元素进行异或即可。 代码 1234567891011int singleNumber(int* nums, int numsSize){ int res = 0; for (int i = 0; i < numsSize; i++) { res ^= nums[i]; } return res;}
